# 🏢 Loghi Clienti

Questa cartella contiene i loghi dei clienti da mostrare nel portfolio.

## Immagini da inserire:

### 1. rcs-logo.png
- **Dimensioni:** 400x400px (1:1)
- **Formato:** PNG con trasparenza
- **Peso:** < 50 KB
- **DPI:** 144 (2x per Retina)
- **Contenuto:** Logo ufficiale RCS Materiali Compositi
- **Usato in:** `SitiWebShowcase.jsx`
- **Importante:** Sfondo trasparente obbligatorio

## Come esportare logo:
1. Richiedi logo vettoriale al cliente (SVG, AI, EPS)
2. Apri in Figma/Illustrator
3. Crea canvas 400x400px
4. Centra logo (lascia 40px padding dai bordi)
5. Esporta PNG 2x (144 DPI)
6. Assicurati sfondo sia trasparente

Per maggiori dettagli vedi: `/AUDIT-IMMAGINI.md`
