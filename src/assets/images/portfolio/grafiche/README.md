# Immagini per GraficheShowcase

Questa cartella contiene le immagini per i 10 progetti del portfolio Grafiche.

## Struttura File

Le immagini dovrebbero essere nominate secondo l'ID del trofeo corrispondente:

1. **branding-logo.jpg** - Logo Tech (Branding)
   - Progetto di logo design con linee guida dell'identità del brand

2. **uiux-icons.jpg** - Icon App (UI/UX)
   - Set di 120 icone personalizzate per app mobile

3. **illustration-editorial.jpg** - Illustration Editorial (Illustration)
   - Serie di illustrazioni editoriali

4. **branding-identity.jpg** - Brand Identity (Branding)
   - Pacchetto completo di identità del brand

5. **print-poster.jpg** - Poster Event (Print)
   - Poster per evento festival musicale

6. **packaging-product.jpg** - Packaging Product (Packaging)
   - Packaging per prodotto alimentare biologico

7. **digital-social.jpg** - Social Media (Digital)
   - Template grafici per social media

8. **editorial-infographic.jpg** - Infographic (Editorial)
   - Infografica animata con visualizzazione dati

9. **illustration-character.jpg** - Character Design (Illustration)
   - Design di personaggio per gioco con turnarounds

10. **uiux-web.jpg** - Web Design (UI/UX)
    - Landing page responsive per SaaS

## Formato Raccomandato

- **Formato**: JPG o PNG
- **Dimensioni**: 1200x1200px (quadrato) per una visualizzazione ottimale
- **Peso**: Max 500KB per immagine (ottimizzare per web)
- **Qualità**: Alta risoluzione ma compressa

## Note

Le immagini verranno utilizzate in due contesti:
1. **Gallery rotante**: miniature che ruotano in 2D sullo schermo principale
2. **Modal dettaglio**: immagine grande e draggabile quando si apre il modal
